The python file below is a simple no-url and easy API  way to download youtube video in the required format. For this you can get the y2mate_api package from the:
pip install y2mate_api
make sure about the dependencies and pip. So the file has an step wise step to get the video or mp3 in the required quality for easy reproduction. The video will be saved either in the downloads or the desktop or the place where your python files are located.
During the video download the progress bar will show the progress.
